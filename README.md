# Portfolio

Welcome to my portfolio website! This project showcases my skills and experiences as a frontend developer.



## Introduction

This is my personal portfolio website. It provides an overview of my skills, projects, and contact information. The website is designed to be simple and easy to navigate.

## Features

- Responsive design
- About me section
- Skills section
- Contact information
- Social media links
- Downloadable resume

## Technologies Used

- HTML
- CSS
- JavaScript
- Font Awesome for icons
- Boxicons for additional icons
- Typed.js for typing animation

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

You need to have `git` installed on your local machine. You can download it from [here](https://git-scm.com/).

### Installation

1. Clone the repository
    ```sh
    git clone https://github.com/Charan26-git/Charan26-git.git
    ```
2. Navigate to the project directory
    ```sh
    cd Charan26-git
    ```

## Usage

Open `index.html` in your preferred browser to view the website.

### Customization

- To update your profile image, replace the `profile.png` file in the root directory.
- Update the `styles.css` file to customize the styling of the website.
- Edit the `script.js` file to add or modify JavaScript functionalities.
